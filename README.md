
# Fluorite

__Create your own styles, simply__

Fluorite is a collection of undogmatic, re-usable, composable set of CSS files for any
project. Copy just the files you need into your project. Put more simply:

> Take what you want, leave the rest.

## Version

The current version is 0.0.0. This project uses [semantic versioning](http://semver.org).

## Features

* Shades of gray by 4% or 5%

## Installation

* Download the zip and extract it into a temp directory
* Copy each file you want into your website theme's css directory
